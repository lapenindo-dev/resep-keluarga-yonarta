# Changelog v2.2.7 — Mode Lihat/Edit Lebih Jelas + Grid 2 Kolom

Perubahan:
- Halaman Detail Resep diberi banner **Mode Lihat Resep** agar berbeda jelas dari mode edit.
- Halaman Tambah/Edit Resep diberi banner mode:
  - **Mode Tambah Resep** untuk resep baru.
  - **Mode Edit Resep** untuk perubahan resep lama.
- Tombol di detail diubah menjadi **Masuk Mode Edit** supaya tidak membingungkan.
- Daftar resep di halaman resep dan resep terbaru di beranda dibuat grid 2 kartu per baris di HP.
- Pada layar lebih besar, kartu resep bisa tampil 3 kolom agar lebih hemat tempat.
- Cache/version naik ke v2.2.7.

Tidak perlu SQL migration baru.
