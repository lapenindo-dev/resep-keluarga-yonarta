# CHANGELOG v3.4.0 — Clean Competitor UI Alignment

Fokus update ini adalah merapikan seluruh fondasi visual agar lebih clean seperti aplikasi referensi (Recipe Keeper / CookBook style direction) tanpa menyalin 100%, sambil tetap mengikuti Master Build Volume 1–3.

## Perubahan utama
- UI di-reset ke arah light, clean, compact, dan image-first.
- Hero home dibuat lebih ringan dan tidak terasa seperti dashboard lama.
- Recipe card dibuat lebih rapat, lebih modern, dan lebih visual.
- Border tebal dikurangi drastis; spacing antar card dirapikan.
- Detail resep dibuat lebih clean dengan foto besar, meta ringkas, dan section yang mudah dibaca.
- Family Hub dibuat seperti side sheet modern yang lebih sederhana.
- Bottom navigation dibuat clean floating bar dengan aksen aktif yang ringan.
- Tetap mempertahankan struktur Master Build: 5 bottom nav, Family Hub, capture-first add flow, dan positioning Family Recipe Heritage App.
