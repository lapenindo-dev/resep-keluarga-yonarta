# Changelog v2.2.4 — Label Input dan Border Foto Utama

- **Foto Masakan** diubah menjadi **Foto Utama**.
- **Foto Resep / Tambahan** diubah menjadi **Foto Tambahan**.
- **Bantu Isi Resep** diubah menjadi **Input Resep**.
- Input Resep disederhanakan menjadi 2 cara:
  1. Input Manual
  2. Foto / Caption
- Panel Foto dan Teks/Caption digabung dalam satu tab **Foto / Caption**.
- Foto Utama diberi border putih halus, outline tipis, rounded corner, dan shadow agar tampil lebih manis serta tidak terlalu mepet ke lebar device.
- Cache/version naik ke **v2.2.4**.

Tidak perlu SQL migration baru.
