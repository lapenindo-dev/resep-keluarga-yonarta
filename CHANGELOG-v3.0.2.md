# Resep Keluarga v3.0.2

## Fokus Update

Perbaikan alur Tambah Resep agar sesuai penggunaan nyata: pengguna lebih sering mulai dari foto resep di kertas, screenshot, chat, atau caption.

## Perubahan

- Memindahkan modul Foto / Caption ke bagian paling atas halaman Tambah Resep.
- Menjadikan Foto / Caption sebagai tab default, bukan Input Manual.
- Menambahkan pilihan asal resep untuk ekstraksi dari foto: Keluarga, Warisan, Internet, Teman, Kreasi sendiri.
- Mengubah copy menjadi lebih jelas untuk foto resep dari kertas lama, tulisan tangan, screenshot chat, dan caption.
- Input manual tetap tersedia sebagai opsi kedua.
- Asal resep tetap tersedia setelah quick capture untuk menjaga konteks keluarga.
- Update version/cache ke v3.0.2.

## Catatan

AI tetap hanya membantu merapikan. User tetap bisa review/edit sebelum menyimpan resep final.
